# Kindle Review Sentiment Analysis

### Sentiment Analysis using Machine Learning

A machine learning project that analyzes **customer reviews from Kindle products** and classifies them as positive or negative sentiment.

This project demonstrates how text data can be processed and transformed into numerical features to build effective sentiment classification models.

---

## Features

### Text Preprocessing
* Tokenization  
* Stopwords Removal  
* Stemming / Lemmatization  

### Feature Engineering
* Bag of Words (BOW)  
* TF-IDF  
* Word2Vec (if used)  

### Machine Learning Models
* Logistic Regression  
* Naive Bayes  
* Other classification models (if used)

## Project Workflow

1. Load Kindle review dataset  
2. Clean and preprocess text  
3. Convert text into numerical features  
4. Train sentiment classification models  
5. Evaluate model performance  
6. Predict sentiment of new reviews
