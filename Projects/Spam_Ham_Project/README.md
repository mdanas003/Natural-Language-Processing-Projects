# Spam Ham Classification

### Email/SMS Spam Detection using Machine Learning

A machine learning project that classifies messages as **Spam or Ham (Not Spam)** using text preprocessing, feature extraction, and classification algorithms.

This project demonstrates how raw text data can be transformed into meaningful numerical features and used to build effective classification models.

---

## Features

### Text Preprocessing
* Tokenization  
* Stopwords Removal  
* Stemming / Lemmatization  

### Feature Engineering
* Bag of Words (BOW)  
* TF-IDF  

### Machine Learning Models
* Naive Bayes  
* Logistic Regression  
* Other classification models (if used)

## Project Workflow

1. Load SMS/Email dataset  
2. Clean and preprocess text  
3. Convert text into numerical features  
4. Train classification models  
5. Evaluate performance  
6. Predict whether a message is spam or not
