# Natural Language Processing Projects

### Text Processing + Feature Engineering + NLP Applications

A comprehensive repository covering **core Natural Language Processing (NLP) techniques**, including text preprocessing, feature extraction, and real-world NLP projects such as spam detection and sentiment analysis.

---

## Features

### Text Preprocessing
* Tokenization  
* Stemming  
* Lemmatization  
* Stopwords Removal  
* Parts of Speech (POS) Tagging  
* Named Entity Recognition (NER)  

### Feature Engineering
* Bag of Words (BOW)  
* TF-IDF  
* Word2Vec  

### NLP Projects
* Spam vs Ham Classification  
* Kindle Review Sentiment Analysis

## Project Workflow

1. Load raw text data  
2. Perform preprocessing (tokenization, cleaning, etc.)  
3. Convert text into numerical features  
4. Train machine learning models  
5. Evaluate model performance  
6. Apply models to real-world NLP tasks
